# Sorting Visualiser 
An application that visualises different kinds of sorting algorithms.

## Technologies Used

 - HTML5
 - CSS
 - JavaScript
 - JQuery
 - NodeJS

## Setup

Then, to start the website, run this executable file:

```bash
$ ./run
```

or run these commands:

```bash
$ npm install
$ npm start
```

this will start the server.

## Usage

To visit the server, go to:

```
localhost:8080
```
